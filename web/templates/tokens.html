Corporation Overview

Token columns:
Configured - Character ID
The current token selected for the corp, this one will be used to try and get wallet data.
Last Journal
The last time the wallet data for this corporation was successfully updated. Wallet data can only be updated with a configured, still valid, active key. ESI only keeps wallet data of the last 30 days, so it is not possible to fix older missed data.
Active
Controls wallet journal updates, if disabled the corporations wallets will not be tracked. Tax reports require wallet data, if disabled the corp will also stop receiving tax reports.
Available - Character ID
Available tokens for the corp, currently selected one is mentioned, use select button action to switch.
Last Checked
Last time the token was checked valid.
Tax Data columns:
CEO
Automatically updated through ESI for new corporations, the CEO character always receives ingame tax report evemails.
You can update all CEOs here, but this may take a minute:
Owner
Optional main contact character in case it is different from the ingame CEO, like with alt or holding corps/characters. If set this character will also receive ingame tax report evemails.
Edit Owner
Edit this to update optional owner of the corp, value should be a character name. Owner column will update if successful, check the result there.
Alt Corp
Whether or not the corp is an alt corp for someone, this affects which tax rules are applied. You can toggle the checkbox to update. [might change to a corp type selection in future if we need more options (main, alt, anti coup, hauler, holding, executor w/e)]
Taxed
Controls if tax is applied to the corporation. If enabled the corp gets tax balance updates and the ceo / owner receives ingame tax report evemails
Tax Balance
If negative: The total tax the corp still owes Brave
If positive: The amount by which the corporation still has overpaid it's tax
Edit Starting Balance
Allows a manual plus or minus amount for the total balance. Useful for corps that had an tax debt or overpayment left from the previous system. Also useful in case a future edit is needed due to mistakes or update failures.
